# Chaluvadi Jaswanth - Cybersecurity Portfolio

A modern, terminal-style portfolio designed to highlight hands-on experience in Security Operations, Detection Engineering, and Incident Response. Built specifically for a Blue Team focus.

## Features
- **Dark Theme / Hacker UI**: Immersive cyber aesthetic with neon green and blue accents.
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices.
- **Projects Showcase**: Highlights complete SOC detection engineering labs, network segmentation labs, and more.
- **Animations**: Smooth page transitions and interactive elements using Framer Motion.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Local Deployment

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/jaswanth/jaswanth-cyber-portfolio.git
   cd jaswanth-cyber-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
